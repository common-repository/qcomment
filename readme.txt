== Description ==

Плагин QComment дает неограниченные возможности для пиара и продвижения в Интернете.
Продвижение брендов, группы в социальных сетях, видео на YouTube, поднять репутацию в Интернете с комментариями и обратной связи.

== Installation ==

1) Разархивировать файл в папке wp-content/plugins
2) Для активации подключаемого модуля необходимо зайти на страницу списка плагинов (Плагины -> Установленные). Найти плагин с названием QComment и нажать кнопку ( Активировать ).
3) Заполнить форму на странице настроек (Настройки -> QComment). В обязательном порядке должен быть указан ключ доступа к API, другие настройки - в зависимости от требований проекта.
4) На создание (редактирование) сообщений будет дополнительный блок (мета коробка), которая позволяет создать проект, купить, принять или отклонить комментарии.
